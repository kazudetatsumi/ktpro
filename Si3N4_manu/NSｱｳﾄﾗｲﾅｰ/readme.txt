ＮＳアウトライナー Readme
----------------------------------------------------------

ＮＳアウトライナーはフリーウェアです。

インストール：
　Setup.exeを実行してください。

アンインストール：
　コントロールパネルの「アプリケーションの追加と削除」にて
NSｱｳﾄﾗｲﾅｰを選択し、「追加と削除」ボタンを押してください。

最新版：
　http://www.nsd.co.jp/share/
　をご覧下さい。

使い方：
　ヘルプを参照ください。
　初めて御使用になる方はヘルプメニューのは「チュートリアル」
をご覧下さい。
